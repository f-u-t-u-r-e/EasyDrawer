## 核心竞争力

相比直接调用SD API，这个Agent的优势：

1. **提示词自动优化**: 
   - 用户输入: "一只猫"
   - Agent输出: 专业的60+词提示词（光影/质感/构图）

2. **参数自适应**:
   - 人像: Steps=30, CFG=7.0, DPM++
   - 风景: Steps=40, CFG=9.0, Euler a
   - 自动选择最优参数组合

3. **质量保证**:
   - 生成3张候选
   - AI自动评分
   - 只返回最佳结果

4. **可扩展架构**:
   - LangGraph支持人工审核
   - Checkpoint断点续传
   - 模块化易于定制
